# Windows IME Cursor Indicator (한영 상태 표시기)

Windows 환경에서 현재 입력 모드(한글/영어)를 마우스 커서와 텍스트 커서 옆에 실시간으로 표시해 주는 유틸리티입니다. AutoHotkey v2로 작성되었습니다.

## 💡 개발 배경 (Motivation)
타이핑을 시작할 때 현재 키보드가 한글 상태인지 영어 상태인지 알 수 없어, 잘못 입력한 글자를 지우고(Backspace) 다시 입력해야 하는 번거로움이 빈번하게 발생합니다.
이 프로그램은 현재 입력 상태를 시각적으로 직관적으로 보여주어 이러한 불편을 해소하고 오타를 방지하기 위해 개발되었습니다.

## ✨ 주요 기능 (Features)

* **실시간 상태 표시 (Dual Display):**
    * **마우스 커서:** 마우스 포인터 우측에 항상 현재 상태가 표시됩니다.
    * **텍스트 커서 (Caret):** 메모장, 브라우저 등에서 글자를 입력 중일 때 텍스트 커서 우측에도 상태가 표시됩니다.
* **직관적인 색상 구분:**
    * **한글 모드:** 파란색 배경 / '한' 표시
    * **영어 모드:** 주황색 배경 / 'A' 표시
* **고해상도 및 듀얼 모니터 지원:**
    * DPI 스케일링이 적용된 환경이나 다중 모니터 환경에서도 좌표 오차 없이 정확한 위치에 표시됩니다.
* **강력한 호환성:**
    * 메모장 등 일반 프로그램뿐만 아니라 Chrome, Edge, VS Code 등 웹 기반/Electron 앱에서도 정상 작동합니다.

## 💻 사용 환경 (System Requirements)

이 프로그램은 다음 환경에 최적화되어 있습니다.

* **OS:** Windows 10 / 11 (64-bit)
* **IME:** Microsoft 입력기 **최신 버전** (New Microsoft IME)
    * *주의: '이전 버전의 Microsoft IME(Legacy)' 호환 모드에서는 동작하지 않을 수 있습니다.*
* **Platform:** AutoHotkey v2.0 이상

## 🚀 사용 방법 (Usage)

1.  [AutoHotkey v2.0](https://www.autohotkey.com/)을 설치합니다.
2.  `KORENG.ahk` 파일을 다운로드합니다.
3.  파일을 더블 클릭하여 실행합니다. (관리자 권한을 자동으로 요청할 수 있습니다.)
4.  실행 시 마우스와 텍스트 커서 옆에 상태창(GUI)이 나타납니다.

### 자동 실행 등록
윈도우 시작 시 자동으로 실행되게 하려면:
1.  `Win + R`을 눌러 실행창을 엽니다.
2.  `shell:startup`을 입력하고 확인을 누릅니다.
3.  열린 폴더에 `KORENG.ahk` 파일의 **바로 가기**를 넣어주세요.

### 종료 방법 (Exit)
* **단축키:** `Shift` + `Esc` 키를 누르면 즉시 종료됩니다.
* 또는 트레이 아이콘을 우클릭하여 종료할 수 있습니다.
